# AGENTS

When writing or revising public-facing text for this project, reference `docs/text-design-guide.md` first. It defines the preferred site title, subtitle, naming tone, atmosphere words, login copy direction, and image-search taste for KoiMoe.

When an author name is needed in copy, examples, metadata, profile UI, or seeded content, use `莉莉姆`.

For visual design, layout, colors, surfaces, and Sakurairo theme fidelity, reference `docs/visual-atmosphere.md`.

This project may be edited by multiple Codex sessions at the same time. A single session may see changes made by another session; treat those changes as external collaborator work, keep them intact, and do not revert or overwrite them unless the user explicitly asks for that exact change.

Avoid changing unrelated Go code or generated assets while updating documentation or copy.
